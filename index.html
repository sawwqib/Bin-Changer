How to Use Bin Changer

Follow these simple steps to transform your Recycle Bin in minutes

1

Download the Setup

Get the ThemeChanger.ps1 by clicking the download button above

2

Run The file

Right Click And run With PowerShell

3

Open Desktop Icon Settings

Right-click on your desktop, select "Personalize", then "Themes", and finally "Desktop icon settings"

4

Select Theme

Type the theme number you want to apply

5

Enjoy Your New Look

enjoy your new Recycle Bin icon!
